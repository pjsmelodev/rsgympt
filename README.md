# RSGymPT

## 📋 Description
RSGymPT is a console-based application developed in C# to manage gym services such as user requests and personal trainer sessions. This project is part of an educational exercise focused on Object-Oriented Programming principles. 

The application allows gym users to:
- Request personal trainer sessions.
- Manage their requests (register, edit, or cancel).
- View and interact with personal trainer schedules.
- Perform administrative tasks via a user-friendly terminal interface.

---

## ⚙️ Features
- **User Management**: Add, remove, and view registered gym users.
- **Personal Trainer Management**: Search and view available personal trainers.
- **Request Management**: Handle the scheduling, cancellation, and completion of requests for personal trainer sessions.
- **Persistence**: All data is stored in JSON files for reliability and portability.

---

## 🛠️ Technologies Used
- **Language**: C# (.NET 8.0)
- **Development Environment**: Visual Studio
- **Data Storage**: JSON files for persistent storage
- **Version Control**: Git and GitHub for repository
